# NexusViewPro

**NexusViewPro** is a lightweight Python package that allows you to render websites and YouTube videos directly inside Jupyter Notebook (`.ipynb`) environments.

It is designed for data scientists, AI engineers, educators, and developers who want to integrate live web content into notebooks for demonstrations, documentation, tutorials, and interactive workflows.
